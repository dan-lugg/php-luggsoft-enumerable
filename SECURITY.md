# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by emailing **dan.lugg@gmail.com**.

Do not open public issues for security vulnerabilities.

You should receive a response within 48 hours. If you do not receive a response, please follow up to ensure your message was received.

## Disclosure Policy

When a vulnerability is reported, we will:

1. Confirm receipt within 48 hours.
2. Investigate and determine impact and scope.
3. Develop and test a fix.
4. Release a patched version and disclose the vulnerability publicly after the fix is available.
