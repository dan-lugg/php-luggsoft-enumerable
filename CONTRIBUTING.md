# Contributing

Thank you for considering contributing to `luggsoft/enumerable`.

## Getting Started

1. Fork the repository.
2. Clone your fork: `git clone git@github.com:your-username/luggsoft-php-enumerable.git`
3. Install dependencies: `composer install`
4. Create a feature branch: `git checkout -b feature/my-feature`

## Development

### Running tests

```bash
composer test
```

### Static analysis

```bash
composer analyse
```

### Code style

Check style:

```bash
composer cs
```

Fix style automatically:

```bash
composer cs-fix
```

## Pull Request Process

1. Ensure all tests pass and static analysis is clean.
2. Add tests for any new functionality.
3. Update docblocks for any changed or new functions.
4. Make sure your code follows the project's coding style (run `composer cs-fix`).
5. Open a pull request with a clear description of the changes.

## Code of Conduct

Please be respectful and constructive in all interactions.
